## Overview

Exam Cell Automation is a robust PHP-based application that automates and simplifies the entire examination lifecycle management process. It provides different dashboards and features for administrators, teachers, and students to manage exams efficiently.

## Features

### Admin Features
- **Manage Students** - Add, edit, and remove student records
- **Student List** - View comprehensive student database with filtering
- **Seating Arrangement** - Automated seating arrangement generation for exams
- **Exam Schedule** - Create and manage exam schedules
- **Question Paper Management** - Upload and organize question papers
- **Result Upload** - Upload student exam results in bulk
- **Result Viewing** - View and analyze exam results
- **Feedback Management** - Review student and teacher feedback
- **Notifications** - Send notifications to students and teachers
- **Reports & Analytics** - Generate detailed reports on exam performance
- **Dashboard Overview** - Quick stats on students, exams, and pending tasks

### Teacher Features
- Access to exam dashboard
- Question paper upload and management
- Result submission
- Student feedback review
- View exam schedules

### Student Features
- User registration and login
- View exam schedules
- Access results and performance reports
- Submit feedback

### General Features
- Secure authentication with password hashing
- Session-based user management
- Role-based access control (RBAC)
- Responsive UI with modern styling
- Mobile-friendly interface
